# Projeto ElasticSearch
Projeto prático com SpringBoot, ElasticSearch e Kibana.

O projeto consistiu na criação de um microservice com SpringBoot para o gerenciamento de dados de pessoas. 

O projeto tem os seguintes serviços:
  - POST: para cadastrar dados de uma pessoa;
  - GET: para consultar uma pessoa;
  - DELETE: para excluir uma pessoa;
  - PATCH: para alterar os dados de uma pessoa.
 
Os dados foram salvos/lidos/excluídos/alterados no ElasticSearch via PostMan.

No Kibana foi criado um dashboard com gráficos baseado nas informações guardadas no ElasticSearch.
